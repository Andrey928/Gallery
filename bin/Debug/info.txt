1. Общие сведения
Программа "Картинная галерея" разработана для учета и управления экспонатами в художественной галерее.
Разработчик: Сушков Андрей, группа ИС-32
Версия: 1.0
Язык реализации: C# (WPF)
СУБД: SQL Server (Entity Framework)
2. Функциональные возможности
Учет произведений искусства (название, автор, год создания, техника, жанр)
Управление выставками (название, даты проведения)
Ведение каталога художников
Учет мест хранения экспонатов
Управление персоналом галереи
3. Руководство пользователя
3.1 Основные разделы
Произведения: Просмотр/редактирование картин
Авторы: Управление художниками
Выставки: Планирование экспозиций
Персонал: Учет сотрудников
3.2 Типовые операции
Добавление: Кнопка "Добавить" → заполнение формы
Редактирование: Выбор записи → "Редактировать"
Удаление: Выбор записи → "Удалить" → подтверждение
4. Техническая информация
Требования к ПО:
.NET Framework 4.7.2+
SQL Server Managment Studio 20
